# 云桌面服务

本目录为云桌面服务相关代码和文档的存放位置。

## 目录结构
- `src/cloud-desktop/docs`: 云桌面相关文档
- `src/cloud-desktop/bkw`: 云桌面客户端代码
- `src/cloud-desktop/cloud-desktop-service`: 云桌面端服务端代码
- `src/cloud-desktop/cloud-desktop-client`: 云桌面客户端代码
- `src/cloud-desktop/proxy`: 云桌面代理服务代码
- `src/cloud-desktop/bkcds-backend`: 云桌面后台调度服务代码
