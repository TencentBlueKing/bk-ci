# IAM 集成、OAuth2 与排查

## 两条常见子链路

- IAM 集成链路：资源同步、实例查询、权限申请、回调
- OAuth2 链路：客户端、令牌、授权码和范围控制

## IAM 方向排查重点

- 资源和动作在 IAM 是否存在
- 回调接口是否返回正确资源结构
- Auth 本地数据与 IAM 外部状态是否一致

## OAuth2 方向排查重点

- 客户端配置是否正确
- 授权范围与资源访问是否匹配
- 令牌生命周期和刷新流程是否符合预期

## 常见坑

- 把 IAM 外部配置问题误判成 Auth 本地服务问题
- 把 OAuth2 认证问题和 RBAC 授权问题混在一起
- 同时改动多条链路，没有先缩小到单一入口
