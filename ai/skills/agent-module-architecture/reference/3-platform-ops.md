# 平台支持、运维与排查

## 平台差异要优先看什么

- 守护方式是否不同
- 路径、权限、进程管理方式是否不同
- 组件安装和升级方式是否不同

## 运维排查顺序

1. Agent 进程是否在线
2. 守护是否正常
3. Ask / 心跳是否持续成功
4. Worker 是否可被正常拉起
5. 升级或清理任务是否把环境破坏了

## 常见平台问题

- Linux 下锁文件、权限或容器环境问题
- Windows 下服务模式与进程模式差异
- macOS 下路径和权限约束

## 边界提醒

- 调度策略不在 Agent
- 插件字段定义不在 Agent
- 任务执行细节优先看 Worker
