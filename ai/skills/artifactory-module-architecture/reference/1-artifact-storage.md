# 制品存储基础

## Artifactory 管什么

`Artifactory` 模块负责：

- 构建产物归档
- 文件元数据管理
- 下载访问
- 报告文件和自定义目录管理
- 后端存储适配

## 常见制品类型

常见会区分：

- 流水线产物
- 自定义目录文件
- 报告文件

这些类型通常影响：

- 路径规则
- 元数据归类
- 查询与下载方式

## 路径心智

制品路径不是普通字符串，而是带有明确业务语义：

- 项目维度
- 流水线维度
- 构建维度
- 任务维度（如报告）

一旦路径规则变化，往往会连带影响：

- 查询
- 下载
- 清理
- 历史构建回看

## 元数据与真实文件

排查时一定要区分：

- 数据库里的文件记录
- 文件任务状态
- 后端存储里的真实文件

三者任何一层出问题，用户看到的现象都可能是“文件不见了”。
