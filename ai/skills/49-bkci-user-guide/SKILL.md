# BK-CI 用户使用指南

## 概述

本 Skill 提供 BK-CI（蓝盾）的完整用户使用指南，帮助用户从零开始掌握 BK-CI 的各项功能。

## 适用场景

当用户遇到以下情况时使用本 Skill：

### 基础使用
- 初次接触 BK-CI，需要了解基本概念和架构
- 不知道如何创建项目或配置权限
- 需要接入代码库或配置触发器
- 想要创建第一条流水线

### 功能操作
- 如何配置构建环境和执行构建任务
- 制品库的使用方法和版本管理
- 质量门禁和代码检查的配置
- 通知服务和审批流程的设置

### 问题解决
- 构建失败或流水线异常
- 权限配置问题
- 日志查看和性能监控
- 常见错误的排查方法

### 最佳实践
- 流水线设计的最佳实践
- 环境变量和参数化构建
- 制品管理策略
- 监控和运维建议

## 使用方式

### 🎯 针对性查阅
根据用户的具体问题，直接引导到相应章节：

```
用户问题类型 → 推荐章节
├── 不了解 BK-CI → 01-基础概念与架构
├── 权限配置问题 → 02-项目与权限管理  
├── 代码库接入 → 03-代码库接入与管理
├── 流水线创建 → 04-流水线创建与配置
├── 构建环境问题 → 05-构建环境与执行
├── 制品管理 → 06-制品管理与发布
├── 质量门禁 → 07-质量红线与代码检查
├── 通知审批 → 08-通知与审批管理
├── 日志监控 → 09-监控与日志管理
└── 故障排查 → 10-故障排查与问题解决
```

### 🚀 快速入门路径
为新用户提供学习路径：
1. 先读 01-基础概念 了解整体架构
2. 再读 02-项目权限 创建第一个项目  
3. 然后读 04-流水线创建 建立第一条流水线
4. 根据需要查阅其他专题章节

### 🔧 问题解决流程
遇到问题时的查阅顺序：
1. 先查 10-故障排查 寻找常见问题
2. 再查相关功能章节了解正确操作
3. 最后查 09-监控日志 分析具体原因

1. **版本兼容性**: 本指南基于 BK-CI 最新版本，部分功能可能在旧版本中不可用

3. **环境差异**: 不同部署环境的配置可能有所差异
4. **最佳实践**: 建议结合项目实际情况调整配置和流程
